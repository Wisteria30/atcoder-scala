# AtCoder Scala環境

## Usage

```sh
cd atcoder-scala/src/main/scala/
# download
acc add abc101
# test
oj t -c "scala Main.scala" -d ./tests/
```

## Requirements

- scala
- atcoder-cli
- online-judge-tools
